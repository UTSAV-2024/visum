# Scoring Rubric

## Overview

Each of the 8 checks is scored independently and contributes to the total score (out of 100). The final band is determined by the total score.

## Checks

1. **AI Bot Permissions (robots.txt)** — Evaluates whether AI agent crawlers are allowed or disallowed
2. **JSON-LD Structured Data** — Checks for structured data markup
3. **llms.txt File** — Checks for the presence of an llms.txt file
4. **MCP Endpoint** — Checks for Model Context Protocol endpoint
5. **JavaScript Rendering** — Evaluates how well the site renders with JS disabled
6. **Meta Tags & Open Graph** — Checks for proper meta tags and OG tags
7. **Sitemap.xml** — Checks for sitemap presence and validity
8. **Page Load Speed** — Measures page load performance

## Bands

| Score Range | Band | Description |
|-------------|------|-------------|
| 80–100 | Agent-Ready | Fully visible to AI agents |
| 60–79 | Partially Visible | Some issues need fixing |
| 30–59 | Mostly Invisible | Significant gaps |
| 0–29 | Agent-Invisible | Barely visible to AI agents |
